# Thư mục ảnh sản phẩm

## Cách thêm ảnh:

1. Đặt 4 ảnh sản phẩm vào thư mục này với tên:
   - `anh1.jpg` (hoặc .png)
   - `anh2.jpg` (hoặc .png)
   - `anh3.jpg` (hoặc .png)
   - `anh4.jpg` (hoặc .png)

2. Định dạng hỗ trợ: `.jpg`, `.jpeg`, `.png`

3. Khuyến nghị:
   - Kích thước ảnh: 600x600px trở lên
   - Tên file không có dấu tiếng Việt (dùng anh1.jpg thay vì ảnh1.jpg)
   - Nén ảnh để tải nhanh hơn

## Lưu ý:

- Nếu chưa có ảnh, trang web sẽ hiển thị placeholder tự động
- Sau khi thêm ảnh, refresh trình duyệt để xem thay đổi
